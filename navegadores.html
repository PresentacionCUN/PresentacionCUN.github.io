Navegadores más utilizados

En ocasiones se suele confundir el término buscador con el de navegador de Internet, esto sucede cuando la página de inicio del navegador es el buscador Google. Un navegador o explorador web, es un programa instalado en nuestro ordenador o dispositivo móvil (smartphone, tableta, ordenador portátil, etc.) para acceder a cualquier sitio web en Internet.

El navegador es el programa que utilizamos para acceder a una página web, un buscador es una página web en la que buscamos información. Los navegadores web más conocidos son: Mozilla Firefox, Google Chrome, Opera, Safari, Microsoft Edge.
